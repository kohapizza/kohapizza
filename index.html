Project

express

"express" is a development environment and visualization tool for the multi-lingual CCG parser with DTS representations "lightblue". This is developed using the Haskell web application framework Yesod, and provides an interactive web interface that lets users explore parsing results, expand or collapse syntactic structures, and better understand how the parser analyzes sentences.

Fat Cat

fat cat is an AR iOS app designed to gently encourage users to go to places they find stressful—like school or work—by placing an adorable virtual cat waiting just for them. Users can set a specific location and time, and when they approach the designated area (within 1km), the cat appears in AR, accompanied by soft background music. They can feed the cat with virtual dried fish, and the cat gradually grows over time, celebrating the user’s effort.

Image Processing Application for Wallpaper Creation

This application generates unique wallpaper outputs from input images each time. The program implements fundamental image processing techniques such as binarization, gamma correction, scaling, and rotation, along with advanced image processing programs like K-Means clustering, chroma key synthesis, Virtual Studio, selective composition, and random composition.
